Event Tracker Web Application
A simple and interactive Event Tracker web application built using PHP, HTML, CSS, JavaScript, Bootstrap, Ajax, and MySQL. The app allows users to manage events, including adding, editing, and deleting events.

Features
Add new events 🆕

Edit existing events ✏️

Delete events ❌

View events by day, week, or month 📅

Responsive design with Bootstrap 📱💻

Technologies Used
Frontend: HTML, CSS, Bootstrap, JavaScript, jQuery, FullCalendar

Backend: PHP, MySQL

AJAX: For seamless updates
